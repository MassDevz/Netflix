🎬 Netflix Clone
📌 Overview
This is a Netflix Clone website built using HTML, CSS, and JavaScript. It replicates the UI of Netflix, including the homepage, movie sections, and responsive design.

🚀 Features
  ✅ Responsive Design – Works on all screen sizes (mobile, tablet, desktop)
  ✅ Movie Sections – Categorized movies like Netflix (Trending, Popular, Originals, etc.)
  ✅ Custom Hover Effects – Interactive movie cards with hover animations
  ✅ Search Functionality – Search for movies dynamically (if implemented)
  ✅ Dark Mode & UI Styling – Sleek Netflix-style dark theme

🛠️ Technologies Used
  HTML – Structure of the webpage
  CSS – Styling and animations
  JavaScript – Dynamic features
